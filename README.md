# ✨ LoveRKN — разблокировка Roblox и связанных сервисов ✨

LoveRKN — это форк проекта Flowseal/zapret-discord-youtube, адаптированный специально для разблокировки Roblox и всех связанных с ним сервисов на территории, где доступ к ним ограничен. Проект помогает обойти ограничения и предоставляет стабильное соединение с игровыми серверами Roblox, а также с сопутствующими сервисами.

## Возможности
- Разблокировка Roblox (клиент, веб-сайт, лаунчер)
- Поддержка связанных сервисов (аутентификация, аватарки, магазин, друзья и т.д.)
- Стабильное соединение с игровыми серверами

## Поддерживаемые сервисы Roblox
- Основной сайт (www.roblox.com)
- Игровой клиент
- API (api.roblox.com)
- CDN (rbxcdn.com)
- Аутентификация (auth.roblox.com)
- Экономика (economy.roblox.com)
- Чат (chat.roblox.com)
- Группы (groups.roblox.com)
- Форум (forum.roblox.com)

## Безопасность
- LoveRKN не собирает пользовательские данные
- Все соединения шифруются
- Открытый исходный код для проверки
- Регулярные обновления актуальности
